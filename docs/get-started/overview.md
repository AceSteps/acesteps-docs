# Overview

Welcome to **AceSteps** - the AI-powered music creation and tokenization platform built on Base Network.

## What is AceSteps?

AceSteps is a Farcaster Mini App that enables users to:

- **Create music** using AI-powered prompts
- **Mint songs as NFTs** (ERC-721)
- **Enable fractional ownership** via ERC-20 tokens
- **Trade song shares** on Uniswap V4
- **Earn revenue** from ad distribution on-chain

## Key Features

| Feature | Description |
|---------|-------------|
| AI Music Generation | Create unique music using text prompts powered by ACE-Step AI |
| NFT Minting | Every song becomes an ERC-721 NFT on Base |
| Tokenization | Enable trading by creating ERC-20 fractional tokens |
| DEX Trading | Trade song tokens on Uniswap V4 pools |
| Revenue Sharing | Earn from ad revenue distributed on-chain |

## Built on Base

AceSteps is deployed on **Base Network** (chainId: 8453), an Ethereum L2 built by Coinbase. This provides:

- Low transaction costs
- Fast finality
- Ethereum security
- Growing ecosystem
