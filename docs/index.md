## What Can You Do?

| Feature | Description |
|---------|-------------|
| **Create Music** | Generate unique AI music from text prompts |
| **Mint NFTs** | Save songs as ERC-721 tokens on Base |
| **Tokenize** | Enable fractional ownership with ERC-20 tokens |
| **Trade** | Buy/sell song tokens on Uniswap V4 |
| **Earn** | Receive ad revenue distributed on-chain |
