# Custom Models

:::info Coming Soon
Custom model support is planned for a future release.
:::

## Overview

AceSteps will support custom fine-tuned models for specialized music generation.

## Planned Features

### Model Types

- **Genre-specific** - Trained on specific genres
- **Style-specific** - Mimicking particular production styles
- **User-trained** - Fine-tuned on your own music

### Use Cases

- Create music in niche genres
- Maintain consistent style
- Develop signature sounds

## Current Limitations

Currently, AceSteps uses the base ACE-Step model:

- General-purpose music generation
- Wide genre support
- Consistent quality
