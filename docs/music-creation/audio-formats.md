# Audio Formats

Understanding audio formats and quality in AceSteps.

## Supported Formats

### Output Formats

| Format | Quality | Use Case |
|--------|---------|----------|
| MP3 | 320kbps | Standard playback |
| WAV | Lossless | High quality archive |
| FLAC | Lossless | Audiophile quality |

### Default Output

- **Format**: MP3
- **Bitrate**: 320kbps
- **Sample Rate**: 44.1kHz
- **Channels**: Stereo

## Audio Quality

### Generation Quality

ACE-Step generates high-quality audio:

- Clear instrument separation
- Natural dynamics
- Professional mastering

### Storage

Audio files are stored on IPFS:

- Permanent storage
- Content-addressed
- Decentralized access

## Duration Options

| Duration | Use Case |
|----------|----------|
| 15s | Short loops |
| 30s | Standard tracks |
| 60s | Extended versions |

## Technical Specs

```
Format: MP3
Codec: LAME
Bitrate: 320 kbps CBR
Sample Rate: 44100 Hz
Channels: 2 (Stereo)
Bit Depth: 16-bit
```
