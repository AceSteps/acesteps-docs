# Music Creation Overview

AceSteps uses **ACE-Step AI** to generate unique music from text prompts.

## How It Works

1. **Enter a Prompt** - Describe the music you want to create
2. **AI Generation** - ACE-Step processes your prompt
3. **Preview** - Listen to the generated music
4. **Save** - Mint as an NFT to your wallet

## Supported Styles

- Lo-fi / Chillhop
- Electronic / EDM
- Hip-hop / Trap
- Ambient / Atmospheric
- Pop / Rock
- Classical

## Quality & Duration

- High-quality audio output
- Variable duration support
- Stems available (future feature)
